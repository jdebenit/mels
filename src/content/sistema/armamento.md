# Sistema de Armamento

Para enfrentar la Variable M, Heracles y otras organizaciones deben utilizar equipo hiper-especializado. Las armas convencionales rara vez son suficientes contra un portador activado.

## Resonadores de Fase

Dispositivos que emiten una frecuencia diseñada para interrumpir la conexión de un portador con la red de bioenergía, induciendo nauseas y pérdida de concentración a nivel táctico.
